Senior Design – Smart Parking System 2.0

Project Overview

Smart Parking System 2.0 is an AI-enhanced, vision-based IoT parking management platform designed for campus-scale deployment. The system integrates embedded edge devices, AI-based vehicle detection, cloud-hosted backend services, and a mobile application to provide real-time parking occupancy, navigation, and administrative oversight.

System Architecture

  • Raspberry Pi edge devices performing AI inference using PyTorch
  • Camera-based parking space and license plate detection
  • Dockerized backend services using Spring Boot
  • Reverse proxy configuration for secure campus access
  • MySQL database for persistent storage
  • React Native mobile application for users and administrators

My Role

As part of the hardware organization and component interaction team, I work on embedded system integration, edge device coordination, and ensuring reliable data transfer between Raspberry Pi units and backend services via HTTP protocols. I contribute to system-level debugging, hardware setup, and validation of real-time parking data transmission.

Engineering Skills Developed

  • Edge AI deployment constraints on embedded hardware
  • Hardware–cloud system integration
  • Containerized backend deployment (Docker)
  • Distributed system coordination
  • Collaborative engineering within a 7-member multidisciplinary team
  • Client communication and requirement adaptation

Impact and Contribution

The project advances a prototype into a campus-ready intelligent parking infrastructure. By combining AI vision, IoT sensing, cloud services, and mobile navigation, the system demonstrates scalable smart-city technology capable of reducing search time and congestion while improving user experience.
